# ParaBank UI Testing

This project focuses on **UI test automation for a banking application** using modern QA practices.

## Scope

The following user workflows are covered:

- User registration  
- User login  
- Account overview  
- Money transfer between accounts  

## Testing Approach

- UI automation using **Playwright**  
- Functional test scenarios (positive & negative cases)  
- End-to-end user journey validation  

## Key Highlights

- Realistic banking scenarios  
- Structured test cases derived from Jira user stories  
- Reusable selectors and test logic  

## Goal

Demonstrate the ability to automate **critical business flows in a banking application** using a modern testing framework.

## Tech Stack

- Playwright (UI automation)  
- JavaScript  
- Node.js  

## Environment

- OS: Windows  
- Browser: Chromium (Playwright)
